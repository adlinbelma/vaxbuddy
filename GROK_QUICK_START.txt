IMPORTANT SECURITY NOTES:
═══════════════════════════════════════════════════════════════════════════

🔒 API Key Security:
   • API key is stored ONLY on Vercel (not in code/git)
   • Never paste API key in chat or commit to repository
   • .env.local is in .gitignore (won't be committed)
   • All API calls go through secure backend

🔒 Data Privacy:
   • Images are converted to base64 and sent to Grok Vision
   • Grok Vision processes the image server-side
   • Only extracted text is returned to VaxBuddy
   • Images are not stored on our servers

═══════════════════════════════════════════════════════════════════════════

Grok Vision API Pricing:
  • ~$0.01-0.05 per API call (depends on image size)
  • Typical vaccine photo: ~3 calls (one extraction per detail entry)
  • Example: 100 vaccines = ~$3-15 in API costs

Free/Cheap Alternatives (if cost is concern):
  • Tesseract.js (already in VaxBuddy) - Free, runs locally
